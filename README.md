# Python Basic Programs

This repository contains simple Python programs for beginners.

## Programs Included

### 1. Basic Mathematical Operations

* Takes two numbers as input
* Performs addition, subtraction, multiplication, and division
* Displays the results

### 2. Personalized Greeting

* Takes first name and last name as input
* Combines them into a full name
* Prints a greeting message

## How to Run

1. Make sure Python is installed
2. Run the program using:

```bash
python filename.py
```

## Purpose

This project is created for learning basic Python concepts like input, output, arithmetic operations, and string handling.
